# Aperture-relative merge

Merge may succeed or fail relative to an observer aperture.

Examples:

- a semantic graph merge succeeds, but the public lowered artifact still wants
  one chosen headline
- a provenance-rich observer sees meaningful divergence, while a coarse state
  observer sees no conflict at all

This suggests merge should be studied as observer-relative rather than as one
absolute boolean result.

Questions:

- which observers admit a canonical join?
- which observers only admit enriched merge?
- when is apparent conflict only a property of the chosen aperture?
