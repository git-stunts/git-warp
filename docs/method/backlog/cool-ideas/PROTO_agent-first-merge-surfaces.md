# Agent-first merge surfaces

Agents should never be forced to reason from raw text-conflict markers if the
substrate can provide richer objects.

The target surface is:

- machine-readable obstruction objects
- explicit branch footprints
- conflict class labels
- candidate lowerings
- witness data sufficient for explanation and repair

This is the anti-sludge merge surface. It would directly improve autonomous
repair, code review assistance, and debugger-guided resolution.
