# Conflict distance

If two branch outcomes do not canonically compose, we still want more than a
boolean obstruction. We want a distance-like quantity:

- how far apart are the competing intents?
- how expensive would it be to translate them into a conflict-free common
  future?
- which branch pairs are near-miss composition failures and which are deep
  semantic opposition?

This could become:

- debugger ranking
- conflict triage
- automated suggestion ordering
- future OG / optic enrichment
