# Merge geometry open questions

The current merge-geometry notes open at least five real theory questions.

## 1. What is the canonical merge object?

Is canonical merge best understood as:

- a join
- a pushout
- a least common future
- or different objects in different subcategories?

## 2. What is the smallest useful enriched space?

What is the minimum structure above canonical state that still preserves
genuine incompatibility honestly?

Candidates:

- braid only
- conflict object only
- paired witness object
- full strand-preserving residual object

## 3. When does policy-guided lowering count as collapse?

If a policy picks one public lowering of an enriched merge, has the system:

- canonically collapsed the conflict
- or merely rendered one observer-relative view of unresolved plurality?

## 4. Can obstruction witnesses compose?

If pairwise merge obstructions are explicit objects, what happens for three-way
or N-way merge? Is there a sane obstruction algebra?

## 5. What is the minimal stable lowering witness?

How much witness is actually needed to lower a canonical or enriched merge back
into a stable public artifact without guessing?
