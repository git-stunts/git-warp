# Visualize braid collapse outcomes in warp-ttd

When the admission kernel ships (Design 0017), braid collapse
produces cell-by-cell outcomes: Derived, Plural, Conflict,
Obstruction. warp-ttd could render this as:

- Green cells = Derived (lawful join)
- Blue cells = Plural (irreducible, preserved)
- Orange cells = Conflict (surfaced with witness)
- Red cells = Obstruction (blocked with reason)

Animate the collapse playing out cell by cell. Show the witness
for each decision. This makes the admission architecture *visible*.
