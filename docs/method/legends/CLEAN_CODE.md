# CLEAN_CODE

Historical umbrella legend retained for old references.

`CLEAN_CODE` / `CC` used to cover a wide swath of structural debt. It
is now too broad to be useful as a live filing legend.

For current `bad-code/` work, use the invariant legends instead:

- `HEX`
- `BND`
- `MODEL`
- `CAST`
- `PORT`
- `OWN`
- `SUB`
- `SPEC`

`CC` remains valid only as a historical prefix in older backlog items,
design docs, retros, and audits. New `bad-code/` notes should not be
filed under `CC`.
