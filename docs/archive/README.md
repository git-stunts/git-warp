# Archive Index

This directory holds historical documents that are useful for traceability but
should not be treated as the canonical current docs set.

Examples:

- superseded plans and drafts
- one-off milestone plans
- [archived architectural decision records](adr/README.md)
- audit transcripts
- completed cleanup checklists

If you are learning `git-warp` for the first time, start with
[../README.md](../README.md) instead.
