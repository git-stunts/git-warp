# VISION

A multi-writer graph database that uses Git as its storage substrate.
Conflict-free by design. No coordination required between writers.

Git-warp exists because graph data deserves the same integrity,
replication, and auditability guarantees that Git provides for source
code — without requiring the data to appear as files in the working
tree.

The AION Foundations Series papers provide the mathematical foundation.
This codebase is the reference implementation.
