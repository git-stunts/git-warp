# BACKLOG

Items waiting to be pulled into a cycle.

When an item is selected for work, move its file to `docs/design/cycles/<cycle-name>/` — the backlog file becomes the design doc for that cycle.

## Process

1. Item lives here as `B{number}.md`
2. Pull into cycle → move to `docs/design/cycles/<cycle-name>/B{number}.md`
3. Write failing tests as spec
4. Implement
5. Retrospective in `docs/archive/retrospectives/`
